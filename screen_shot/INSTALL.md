# 快速安装指南

## Chrome 浏览器安装步骤

1. 打开 Chrome 浏览器
2. 在地址栏输入: `chrome://extensions/`
3. 在页面右上角启用 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"** 按钮
5. 在文件选择器中,导航到并选择 `screen_shot` 文件夹
6. 点击 **"选择文件夹"**
7. ✅ 安装完成!扩展图标会出现在浏览器工具栏

## Edge 浏览器安装步骤

1. 打开 Microsoft Edge 浏览器
2. 在地址栏输入: `edge://extensions/`
3. 在页面左下角启用 **"开发人员模式"**
4. 点击 **"加载解压缩的扩展"** 按钮
5. 在文件选择器中,导航到并选择 `screen_shot` 文件夹
6. 点击 **"选择文件夹"**
7. ✅ 安装完成!扩展图标会出现在浏览器工具栏

## 使用方法

1. 访问任意网页
2. 点击浏览器工具栏中的扩展图标(绿色相机图标)
3. 在弹出窗口中点击 **"Capture Full Page"** 按钮
4. 等待截图完成(状态会显示进度)
5. 浏览器会自动弹出保存对话框
6. 文件名已自动填充为 `域名_时间戳.png` 格式
7. 选择保存位置,点击保存即可

## 提示

- 📌 **固定扩展**: 右键点击扩展图标,选择"固定"以便快速访问
- 🔄 **更新扩展**: 如有代码修改,在扩展管理页面点击"刷新"按钮
- ⚙️ **权限**: 首次使用时可能需要授予必要权限
