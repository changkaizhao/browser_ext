# 快速开始指南

## 安装扩展

### Chrome
```
chrome://extensions/ → 开发者模式 → 加载已解压的扩展程序 → 选择 screen_shot 文件夹
```

### Edge
```
edge://extensions/ → 开发人员模式 → 加载解压缩的扩展 → 选择 screen_shot 文件夹
```

## 使用方法

### 📸 完整页面截图
1. 点击扩展图标(绿色相机)
2. 点击 **"Capture Full Page"** (绿色按钮)
3. 等待自动截图完成
4. 选择保存位置

**文件名格式:** `域名_时间戳.png`

---

### ✂️ 区域选择截图
1. 点击扩展图标
2. 点击 **"Select Area to Capture"** (蓝色按钮)
3. 在页面上拖拽鼠标选择区域
4. 释放鼠标完成截图
5. 选择保存位置

**快捷键:**
- **ESC** - 取消选择
- 拖拽时实时显示尺寸

**文件名格式:** `域名_area_时间戳.png`

---

## 故障排除

### ❌ 出现连接错误
**解决方法:**
1. 刷新要截图的页面
2. 重新点击扩展图标

扩展会自动注入必要的脚本。

### ❌ 截图不完整
**可能原因:**
- 页面有懒加载内容
- 动态加载的图片未加载完成

**解决方法:**
先手动滚动到页面底部,等待内容加载完成后再截图。

---

## 提示

✅ 完整页面截图会自动处理滚动
✅ 区域截图只能截取当前视口内的内容
✅ 截图为PNG格式,无损质量
✅ 文件名自动填充,可修改
✅ 支持所有网页(除了浏览器内置页面)
